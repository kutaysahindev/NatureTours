## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

This program has no pre-requisites.

### Installation

1. Clone the repo

```sh
gh repo clone kutaysahindev/NatureTours
```
2. Install NPM packages

```sh
npm install
```

3. Open The index.html file
